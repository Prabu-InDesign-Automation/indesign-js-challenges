InDesign to PDF Export Script (with UI)
Overview

This ExtendScript automates the process of exporting Adobe InDesign documents to PDF. It includes a user-friendly UI that allows users to control export options before generating the final output.

Features
Export active InDesign document to PDF
Simple UI for user interaction
Supports customizable export settings
Automatically handles document selection
Reduces manual export steps and improves workflow efficiency

Developed for InDesign automation practice and workflow optimization.
